# Task 3

## Запуск

`make arg1=PROC arg2=F arg3=L arg4=FILE`, где PROC - число процессов, F и L - соотв. начало и конец диапазона, FILE - имя файла для вывода   

## Тестирование

`make test` - тестирует от 1 до 20 процессов, строит графики по времени
