# Task 3 (pthreads)

## Компиляция

`make`

## Запуск

`./main <THEAD_NUM> <FIRST> <LAST> <OUT_FILE>` - запустить на THREAD_NUM тредах, в интервале от FIRST до LAST, вывод в OUT_FILE   

## Тестирование

`make plot` - тестировать и построить график
